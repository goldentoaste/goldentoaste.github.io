Github repo Deployed demo

A demo for a Svelte component I made, its meant to be a infinite scroller to browse various items. User can swipe up and down like its Tictok, and swipe left or right to accept or reject like Tinder. It's made to be compatible with both web browser and mobile, and can display any content inside. The current implementation is a lil buggy for now, so its only good for browsing cat pictures.
