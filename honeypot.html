Honey pot

Honey pot / Sticky markdown down, was originally started because I wanted a better sticknote program for windows, but I wanted to write markdown instead. Eventually the program was discountinued due to feature creep, python's low performance, and Qt's restrictions. Maybe it just a skill diff.

Some feature included are:

  • Markdown syntax highlighting
  • Live markdown preview
  • Markdown code block syntax highlight for Js and Python
  • Extendable framework for adding additional language support
  • Rebinable keyboard short cuts for editing quick actions
  • Image caching system to paste image directly into document
  • Cache for web image links

Some of these features requires the markdown to be preprocessed before that being passed to Qt to be rendered, and also Qt requires the entire document to be reloaded so it's not hard to be see that performance is bad for larger documents. So ultimately at some point I felt like I was trying to make a proper markdown editor instead of a simple sticky note and so it wasn't worth it anymore.

This project was initially started as a quick n dirty interface to play the game for my game in the games Ai course. But the game turned out t be really fun and so I decided to make the multiplayer work and made it look nice.

Pente's rules are sort of similar to Gomoku, where the player can win instantly if they connected 5 pieces in a row. But it also has a capture rule, where if a player "sandwich" 2 opponent pieces, those 2 inner ones are captured. The capture mechanic makes the game significantly more interesting by making the win condition less obvious for casual players, and makes the game more complex for advance players.

I think I've made one of the better ways to play this game online, with features such as:

  • Online peer to peer multiplayer
  • Local single player
  • Spectators
  • History of game players, both players can rewind to any past plays
  • Chat room for both players and spectators
  • Export games and load previous games
  • Resizable board.
  • Maybe ai opponent in the future :?
