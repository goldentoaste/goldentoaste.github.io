A custom interpreted programming language implemented in python. The general architecture is mostly borrowed from Crafting Interpreter, but I put fair amount of my own spin on it. The language supports basic programming features, functional programming, and basic OOP, with a syntax similar to javascript, and a few ideas borrowed from Dreamberd. Please see demo site or github repo for more some documentation.

Some features:
  • Dynamically typed
  • Functional programming, pass function as variables
  • Functional closure
  • Function overloading
  • Chained comparison
  • C-style variable closure
